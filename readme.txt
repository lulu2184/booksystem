[How to Run]
————————————————————————————————————————————————————————————————-
	1. javac frontend/*.java backend/*.java
	3. In windows system: java -cp "./mysql.jar;." frontend.Controller
 	   In Linux/Unix system: java -cp ./mysql.jar:. frontend.Controller
	4. Or you can use an IDE (such as Ecilipse) as we have demoed in class. 
	   Remember to add mysql.jar as an external jar to your project's Java build path.
